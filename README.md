# MSToolbox

> 微服务工具箱

## 工程结构
``` 
MSToolbox
├── ms-bom -- 统一依赖版本管理
├── ms-core-auto -- Spring 自动配置
├── ms-core-context -- Spring 上下文工具
├── ms-core-db -- 数据库连接类及Druid配置
├── ms-core-launch -- Spring 自定义启动类
├── ms-core-tool -- MS 通用工具
├── ms-actuate-starter -- Spring HTTP 缓存管理
├── ms-api-crypto-starter -- Spring API 数据加密与解密
├── ms-ehcache-starter -- Spring 数据缓存
├── ms-file-storage-starter -- Spring 文件管理
├── ms-http-starter -- Spring HTTP 工具类
├── ms-loadbalancer-starter -- Spring 负载均衡
├── ms-mybtais-encrypt-starter -- MyBatis 处理字段加密
├── ms-redis-starter -- Spring Redis 处理工具
```
